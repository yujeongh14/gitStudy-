깃허브
	원격저장소로 활용되는 서버
	
깃허브 (원격저장소)와 로컬저장소 연결
1. 원격저장소 만들기
	깃허브 로그인 > repository 만들기

2. 내가 만든 원격저장소와 로컬 저장소 연결
	1. git branch -M main
		브랜치를 main으로 변경
	2. git remote add 별칭 원격저장소 주소
		git 로컬저장소와 연결된 원격저장소를 설정
		원격 저장소 주소가 너무 기니까 별칭 사용 (보통은 orgin 사용)
git remote
	해당 레포지토리와 연결된 원격 저장소를 알려줘!
			
git push 별칭 브랜치 이름
	git push origin main 
		--> origin 이라는 별칭을 가진 원격저장소의 main이라는 브랜치에다 업로드 하겠다

-u 옵션
	git push -u 별칭 브랜치이름
	해당 옵션을 사용하면 별칭과 브랜치 이름은 생략이 가능하다
