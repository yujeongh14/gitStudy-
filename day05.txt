git remote add 원격저장소별칭 원격저장소주소
	해당주소의 원격저장소를 별칭으로 추가(연결)
git remote		
	연결시킨 원격저장소 별칭 확인

git remote -v
	연결시킨 원격저장소 별칭과 주소

git remote show 원격저장소 별칭
	원격저장소의 상세 내용 확인(브랜치)

git clone 원격저장소주소
	원격저장소의 커밋내용, 수정사항 등 모든 과정을 복제

----------------------------------------------------------------------------------------------------------------
원격저장소의 내용을 다운로드 하는 방법
git pull
	원격저장소의 최신 커밋상태를 로컬 저장소로 다운로드

git fetch 
	해당주소의 원격저장소 커밋 내용(만)을 복사하여 내려받기

git merge 원격저장소별칭/브랜치이름 생략 가능
	내려받은 로컬저장소에 적용

git remote rm 원격저장소 별칭
	원격저장소의 별칭 삭제

git log 
	커밋 내용(로그)을 확인할 수 있으며 q를 클릭하면 원래대로 돌아옴
	 
	각각의 커밋 내용은 고유의 id값을 갖고 있다
	각각의 id는 중복이 되지 않으므로 그 id로 커밋을 구분한다
